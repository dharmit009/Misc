# Cheatsheet

*NOTE: This is a quick cheatsheet for common markdown it may not include all the features. 

## Headings: 

/# Heading 1
/## Heading 2
/### Heading 3

# Heading 1
## Heading 2
### Heading 3

## Italics: 

/*Italics /*
*Italics.*

## Bold: 

/*/* -> Bold./*/*
**Bold.**

## Inline Links: 

/[Duckduckgo/]/(www.duckduckgo.com)
[Duckduckgo](www.duckduckgo.com)
